FAQ

Questions & answers

Does Foliow require an internet connection? +
The app itself runs 100% offline. The local proxy (folio_proxy.py) needs internet access to fetch live prices from Yahoo Finance and Stooq. Your portfolio data never leaves your machine.
What markets are supported? +
US stocks and ETFs (via Finnhub API — free key), European equities (.PA, .AS, .DE, .MI, .MC, .L and more), crypto (BTC-USD, ETH-EUR…), gold/silver ETCs, and structured products. Essentially any ticker available on Yahoo Finance.
Do I need a Finnhub or API key? +
Not strictly required. Foliow works without any API key using Stooq (EOD prices) for all markets. For real-time US quotes, a free Finnhub key (finnhub.io) takes 2 minutes to set up. The AI Plan tab requires your own Claude API key (Anthropic).
Do I need Python installed? +
Yes — Python 3.8 or later. No additional packages needed; the proxy uses only the Python standard library. If you're on Windows, Python 3.11 from python.org works perfectly.
Is my financial data private? +
Completely. Foliow stores everything in your browser's localStorage — no account, no server, no cloud sync. The proxy only fetches anonymous price data. Your positions, quantities, and P&L are never transmitted anywhere.
Can I use it on multiple computers? +
Yes. Your license is personal and unlimited — install on as many devices as you like. Portfolio export/import (JSON) lets you sync data between machines manually.
